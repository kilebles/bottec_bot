### Запуск проекта

1. Клонировать репозиторий:

   ```bash
   git clone https://github.com/kilebles/bottec_bot.git
   ```
2. Установить зависимости:

   ```bash
   poetry install
   ```
3. Создать `.env` на основе шаблона `.env.template`

4. Запустить сервер:

   ```bash
   poetry run python run.py
   ```
